Image Cropper Install Guide Demo Github

安装

使用bower：

"image-cropper": "git@github.com:ElemeFE/image-cropper.git#~0.1.5"

使用npm：

npm install image-cropper.js --save

引入文件

Image Cropper的使用依赖两个文件：

  • dist/cropper.css
  • dist/cropper.js

最简单的办法是直接在HTML中引用这两个文件，不过每个项目的情况不同，可以根据自己的项目情况来决定如何引入这两个文件。

使用说明

Image Cropper是因为一个裁剪头像并上传的需求诞生的。

一个裁剪头像的流程大概是这样：

  • 用户选择一张本地图片。
  • 用户使用鼠标拖拽裁剪头像，并实时预览不同大小的裁剪效果。
  • 把用户的裁剪范围和图片提交到服务器端。

从提交到服务器端的角度来看，我们关注的点只有两个：

  • 用户选择的图片(File)
  • 用户的裁剪结果(Rect: left、top、width、height)

在Angular中使用

Image Cropper为Angular提供了三个Directive，通过这三个Directive的配合，即可实现一个图像裁剪和预览的效果。

  • cropper: 添加到要显示裁剪框的元素上
  • cropper-source: 添加到input type=file上。
  • cropper-preview: 添加到要显示裁剪预览的元素上。
  • cropper-context：用户的裁剪结果。

Image Cropper在同一个页面上可以指定多个Instance，该属性定义一个字符串，作为Image Cropper的一个实例的id。

cropper

这个Directive，添加到要显示裁剪框的元素上。 显示裁剪框的元素应该是这样：

<div class="image-container source" cropper="avatar" cropper-context="cropContext">
  <img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" class="noavatar"
  />
  </div>

cropper-source

这个Directive需要绑定在一个input上，在定义的时候需要定义Cropper ID。

<input type="file" name="avatar" cropper-source="avatar" />

cropper-file-types

默认支持的类型为gif、png、jpg，如果需要定义只支持某一种或者某几种格式，比如只支持jpg、png，可以这么定义：

<input type="file" name="avatar" cropper-source="avatar" cropper-file-types="jpg,jpeg,png"
  />

cropper-preview

在用户裁剪图像的过程中，需要动态的预览不同大小的头像的效果，需要在预览图像的Container上添加这个directive。

<div class="image-container small" cropper-preview="avatar">
  <img src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" class="noavatar"
  />
  </div>

独立使用

如果要脱离Angular独立使用Image Cropper。

var cropper = new Cropper({
  element: document.getElementById('demo'),
  previews: [ document.getElementById('preview-large') ],
  onCroppedRectChange: function(rect) {
  //write your logic
  }
  });

  cropper.setImage('avatar-big.jpg');
